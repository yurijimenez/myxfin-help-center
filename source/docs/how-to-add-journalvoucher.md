# Journal Voucher

The Journal Voucher (JV) module is used for manual accounting entries, such as depreciation, accruals, or correcting entries that aren't covered by other specific modules.

---

### How to Create a New Transaction

1. Click the **Create New** button.

2. **Select Entity (Optional)**: Use the **Dropdown Menu** at the top left to choose between **Customer**, or **Supplier**. This allows you to search for and link a specific entity to the transaction.

3. **Set the Memo**: Enter a clear description in the **Memo** box to explain the purpose of the journal entry.

4. **Enter Accounting Details**: In the Details table
   **Acct# / Account Title>** Enter the account number or description for the first line.
   **Debit/Credit>** Enter the amount in the appropriate column.
   **Tax/Cost Center>** Select the applicable **EWT/VAT Code** and **Cost Center** if required.

5. **Add New Lines**: To add more rows, press **TAB** after the remarks field of the last row.

6. **Balance Check**: Ensure the **Out of Balance** field shows **0.00**. The system will not allow you to save if the total Debits do not equal the total Credits.

7. **Finalize**: Click the **Save (CTRL+S)** button to post the entry.

---

### How to Edit a Transaction

1. **Open the Voucher** From the main list, click the specific **JV Number**

2. **Modify Lines** You can update amounts, account titles, or remarks. 
   
3. **Save Changes** Click the **Save** button once the entry is balanced again.

* *Note: If a transaction is already posted or reconciled, you may need to unpost it before editing.*

---

### Navigation & Shortcuts

* **Back to Main (ESC):** Use this button or the ESC key to exit without saving.

* **Attachments Tab:** Use this tab to upload supporting documents (like bank notices or invoices) related to the journal entry.

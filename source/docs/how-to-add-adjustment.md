# AP Adjustment

AP (Accounts Payable) Adjustment allows users to correct, update, or reverse vendor-related transactions in MyxFin. This feature is useful for fixing mistakes in bills, payments, or credits, ensuring that your accounts payable records are accurate. Users can create new adjustment entries, edit existing transactions, or unpost incorrect entries to maintain accurate financial reporting.

![Navigating to Adjustments](/assets/img/adjustment.png)
*Figure 1: Adjustment.*

> 💡 **Take note:**Only transac ons without reference or with cancelled reference in Receive Payment will be available for unposting.

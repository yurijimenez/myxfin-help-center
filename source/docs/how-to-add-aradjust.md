# AR Adjustment

AR (Accounts Receivable) Adjustment allows users to correct, update, or reverse customer-related transactions. This ensures your records remain accurate when fixing mistakes in invoices or payments.

> 💡 **Note:** Only transactions without a reference (or with a cancelled reference in Receive Payment) can be unposted.

---

### How to Create a New Transaction

1. **Open the Form** Click the **Create New** button to start a new adjustment.

2. **Choose your Reference Type** > 
For **Sales Returns**: Check the Sales Return box before clicking the search button. This will generate an automatic accounting entry (which remains editable).For **Sales Invoices**: Uncheck the Sales Return box, then click the **Search** button to select your reference Sales Invoice.

3. **Add Line Items** Click the **Add New Line** button in the details section to insert your adjustment entries.

4. **Save the Transaction** Once all details are correct, click the **Save** button to finalize.

![Navigating to AR Adjustment](/assets/img/aradjust2.png) 

---

### How to Edit a Transaction

If you need to modify an existing adjustment, follow these steps:

1. **Activate Edit Mode** Click the **Edit** button. This will unlock the form fields for modification. 

2. **Modify Details** Update the necessary amounts or account titles.

3. **Commit Changes** While in editing mode, the **Save** button will become active. Click it to apply your updates.

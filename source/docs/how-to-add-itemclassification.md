# Item Classification

Item Classification is used to organize and categorize items in MyxFin. This module allows users to create, update, or deactivate classifications to better manage their inventory and reporting. By assigning items to proper classifications, you can easily track product types, generate reports, and maintain a well-structured inventory system.

---

### How to Create a New Classification:
1. Go to**Master Data**Files. 
2. Click**Items**
3. Click**Classification**
4. Click**Create New (F1)**
5. Enter the details of the new Type. 
6. Click**Add Details**

![Navigating to cl](/assets/img/cl1.png)
*Figure 1: Create new classification.*

---

### How to Update Classifications :
  

1. Go to**Master Data**Files. 
2. Click**Items**
3. Click**Classification**
4. Click the**Item classification**code. 
5. Enter new details. 
6. Click **Update** Detail.

![Navigating to cl](/assets/img/cl2.png)

---

### How to Inactive Classifications :

1. Go to**Master Data** Files. 
2. Click**Items**
3. Click**Classification** 
4.  Click**Active (Green)**/**Inactive (Yellow)**icon.

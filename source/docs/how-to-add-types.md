# Item Types

Item Types in MyxFin are used to categorize inventory items based on their nature or purpose. This module helps organize items within the system, making it easier to manage inventory, track transactions, and generate accurate reports.

---

### How to Create a New Types 
1. Go to**Master Data Files** 
2. Click**Items**
3. Click**Type**
4. Click**Create New (F1)**
5. Enter the details of the new Type. 
6. Click**Add Details**

### How to Update Types :

1. Go to**Master Data Files**
2. Click**Items**
3. Click**Type**
4. Click the**Item type code**
5. Enter new details
6. Click**Update** Detail

![Navigating to Banks](/assets/img/types2.png)

---

### How to inactive the Types:

1. Go to**Master Data** Files
2. Click**Items**
3. Click**Type**
4. Click**Active**(Green) /**Inactive** (Yellow) icon

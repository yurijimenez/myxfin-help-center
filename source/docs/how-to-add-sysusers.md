# System Users

System Users define authorized user accounts in the ERP system. Each user is assigned specific module access and action permissions, such as view, add, edit, post, cancel, and print.These settings control what functions a user can access and perform, ensuring proper system control, security, and accountability.

---

### How to Create or Add New User:

1. Go to **Master Data** Files
2. Click **System Users**
3. Click **Create New (F1)**
4. Input the details for the new user. 
5. Click **Add User**

![Navigating to System User](/assets/img/sysuser.png)

---

### How to Set User's Access:

1. Go to **Master Data** Files. 
2. Click **System Users** 
3. Select a user and click the pencil icon or Edit user's access. 
4. Select the **categories** for which the user has access. 
5. Once you've chosen the category, mark the checkbox corresponding to the user's access.   
6. If user has an all access, just tick the **CHECK ALL** 
7. Click **Save (F2)**

![Navigating to System User](/assets/img/sysuser1.png)
![Navigating to System User](/assets/img/sysuser2.png)

---

### How to Reset User Password:

1. Go to **Master Data** Files. 
2. Click **System Users**
3. Click **cycle** icon or **Reset** user password. 

![Navigating to System User](/assets/img/sysuser3.png)

--- 

### How to Inactive User Access:

1. Go to **Master Data** Files. 
2. Click **System Users** 
3. Click **x** icon **Inactive** user access. 

o To activate the inactive user, just click the **check icon** or **Activate** user access.

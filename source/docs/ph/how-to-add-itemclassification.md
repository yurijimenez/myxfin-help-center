# Item Classification

Ang Item Classification ay ginagamit upang ayusin at ikategorya ang mga items sa MyxFin. Sa module na ito, maaaring gumawa ng bagong classification, i-update ang existing classification, o i-inactivate ang mga lumang classification upang mas maayos na pamahalaan ang inventory at reporting.

---

### Paano Gumawa ng Bagong Classification:
1. Pumunta sa  **Master Data** Files
2. I-click ang **Items**
3. I-click ang **Classification**
4. I-click ang **Create New (F1)**
5. Ilagay ang mga detalye para sa bagong Classification. 
6. I-click ang **Add Details**

![Navigating to cl](/assets/img/cl1.png)
*Figure 1: Paggawa ng bagong classification.*

---

### Paano Mag-update ng Classifications:
  

1. Pumunta sa **Master Data** Files
2. I-click ang **Items**
3. I-click ang **Classification**
4. I-click ang **Item classification code**
5. Ilagay ang mga bagong detalye. 
6. I-click ang **Update Detail**

![Navigating to cl](/assets/img/cl2.png)

---

### Paano Gawing Inactive ang Classifications:

1. Pumunta sa **Master Data** Files
2. I-click ang **Items**
3. I-click ang **Classification** 
4. I-click ang **Active (Green)** / **Inactive (Yellow)** icon. 

![cl](/assets/img/cl3.png)

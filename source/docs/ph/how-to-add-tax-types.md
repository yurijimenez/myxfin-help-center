# Tax Types

Ang Tax Types ay ginagamit para sa tamang tax classification na kinakailangan sa pagproseso ng mga transaksyon at sa pag-uulat sa BIR. Ang mga kasalukuyang tax types ay naka-set up na base sa mga requirements ng BIR.

> 💡 **Tandaan:** Gumawa lamang ng bagong tax type kung mayroong karagdagang tax classifications na ipinag-uutos ang BIR sa hinaharap.

--- 

### Paano Gumawa ng Bagong TAX Type:

1. Pumunta sa **Master Data Files** 
2. I-click ang **Accounting Parameters** 
3. I-click ang **TAX Types** 
4. I-click ang **Create New (F1)** 
5. Ilagay ang mga detalye ng TAX Type. 
6. I-click ang **Add Detail**

![Navigating to Taxtype](/assets/img/taxt.png) 

---

### Paano Mag-update ng TAX Type:

1. Pumunta sa **Master Data Files**. 
2. I-click ang **Accounting Parameters** 
3. I-click ang **TAX Types**
4. Pumili ng tax type at i-click ang blue font text o ang TAX Type Code. 
5. Ilagay ang mga bagong detalye ng Tax Type. 
6. I-click ang **Update Detail** 

![Navigating to Taxtype](/assets/img/taxt2.png) 

---

### Paano Gawing Inactive ang TAX Type:

1. Pumunta sa **Master Data Files**
2. I-click ang **Accounting Parameters** 
3. I-click ang **TAX Types**
4. I-click ang **yellow cycle icon** o ang **Inactive** button.

I-click naman ang **green cycle icon** o ang **Active** button para muling i-activate ang item.

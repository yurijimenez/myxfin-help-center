# Sales Invoice

Ang Sales Invoice module ay ginagamit upang mag-bill sa mga customer para sa mga produktong na-deliver o serbisyong naibigay. Ina-update nito ang iyong accounts receivable at direktang naka-link sa mga naunang delivery o billing records.

> 💡 **Tandaan:** Ang mga transaksyon ay maaari lamang i-unpost kung wala itong active references (o may cancelled references) sa **Adjustments** o **Receive Payment**.

---

### Paano Gumawa ng Bagong Transaksyon

1. **Simulan ang Invoice**: I-click ang **Create New** button upang magbukas ng bagong form.

2. **Piliin ang Source Reference**: Depende sa iyong ini-invoice, piliin ang tamang reference type.  
Para sa Services → Piliin ang **Billing** o **Sales Order**.  
Para sa Goods → Piliin ang **Delivery**.

3. **Piliin ang Transaksyon**: I-click ang partikular na **Transaction No.** mula sa listahan upang makita ang available items.

4. **Piliin ang Mga Item**: I-check ang mga checkbox sa kaliwang bahagi ng mga item na nais isama sa invoice, pagkatapos ay i-click ang **Insert**.

5. **I-finalize at I-save**: I-review ang totals at i-click ang **Save** button.

### Paano Mag-edit ng Transaksyon

1. **Pumasok sa Edit Mode**: I-click ang **Edit** button upang ma-unlock ang transaction fields.

2. **Baguhin ang Mga Detalye**:  
**Magbawas ng Item:** I-click ang **Remove/Delete** button katabi ng line item upang alisin ito sa invoice.  
**I-update ang Fields:** Magiging active lamang ang **Save** button habang nasa editing mode.

3. **Suriin ang Accounting**  
Pagkatapos mag-save, maaari mong i-click ang **Accounting Entry** button upang makita ang journal entries na ginawa para sa invoice.

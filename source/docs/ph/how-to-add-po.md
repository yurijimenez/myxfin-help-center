# Purchase Order

Ang Purchase Order (PO) module ay nagbibigay-daan upang pamahalaan ang mga order para sa goods o services mula sa iyong suppliers. Tinitiyak nito na lahat ng pagbili ay naitatala, nasusubaybayan, at naaayon sa mga aprubal.

> 💡 **Tandaan:** Ang mga transaksyon ay maaari lamang i-unpost kung wala itong active references (o may cancelled references) sa **Receiving** o **APV**.

---

### Paano Gumawa ng Bagong Transaksyon

1. **Simulan ang Order**  
I-click ang **Create New** button upang buksan ang bagong form.

2. **Piliin ang Supplier**  
Hanapin at piliin muna ang **Supplier Name**

3. **Piliin ang Payment Type**:  
Credit → Para sa mga pagbiling babayaran pagkatapos ng delivery.  
Advance → Para sa mga pagbiling babayaran bago ang delivery.

4. **Magdagdag ng Items (Dalawang Paraan)**:  
**Manual Entry** → Kung hindi kailangan ang PR, hanapin ang items gamit ang **Code** o **Description** direkta sa details section.  
**Reference (PR/SO)** → I-click ang **Reference (Insert)** button upang kunin ang items mula sa existing Purchase Request o Sales Order.

5. **Pumili mula sa Reference** (Kung ginagamit ang Method 2):  
**I-click** ang partikular na **Transaction No.** sa pop-up window.  
**I-check** ang mga box sa kaliwa para sa items na kailangan mo.  
**I-click** ang Insert.

6. **I-save**  
I-review ang quantities at i-click ang **Save** button.

### Paano Mag-edit ng Transaksyon

> 💡 **Tandaan:** Maaari lamang i-edit ang mga transaksyon na hindi pa fully approved.

1. **Pumasok sa Edit Mode**  
I-click ang **Edit** button upang ma-unlock ang form.

2. **Baguhin ang Mga Detalye**  
I-update ang quantities, payment terms, o items kung kinakailangan.

3. **I-save ang Mga Pagbabago**  
Magiging active ang **Save** button habang nasa editing mode. I-click ito upang ma-finalize ang updates.

### Mga Aksyon sa Dashboard

* **Go Back to List:** I-click ito upang lumabas sa form nang hindi nagse-save.

* **Filtering:** Gamitin ang mga parameters sa main dashboard upang subaybayan ang PO statuses o maghanap ayon sa supplier.

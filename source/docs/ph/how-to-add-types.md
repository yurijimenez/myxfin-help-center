# Item Types

### Paano Gumawa ng Bagong Item Type:
1. Pumunta sa **Master Data Files** 
2. I-click ang **Items**
3. I-click ang **Type**
4. I-click ang **Create New (F1)**
5. Ilagay ang mga detalye para sa bagong Type. 
6. I-click ang **Add Details**

---

### Paano Mag-update ng Type:

1. Pumunta sa **Master Data Files**
2. I-click ang **Items**
3. I-click ang **Type**
4. I-click ang **Item type code**
5. Ilagay ang mga bagong detalye.
6. I-click ang **Update Detail**

![Navigating to Types](/assets/img/types2.png)

---

### Paano Gawing Inactive ang Type:

1. Pumunta sa **Master Data Files**
2. I-click ang **Items**
3. I-click ang **Type**
4. I-click ang **Active** (Green) / **Inactive** (Yellow) icon.

![Navigating to Types](/assets/img/types3.png)

# Purchase Request

Ang Purchase Request (PR) module ay nagbibigay-daan upang humiling ng mga items o serbisyo bago gumawa ng formal na Purchase Order. Tinitiyak nito na ang lahat ng procurement ay nasusubaybayan at naaaprubahan.

> 💡 **Tandaan:** Ang mga transaksyon ay maaari lamang i-void kung wala itong reference (o may cancelled na reference) sa Purchase Order.

---

### Paano Gumawa ng Bagong Transaksyon

1. **Simulan ang Request**: I-click ang **Create New** button upang buksan ang form.

2. **Itakda ang Section**: Piliin ang tamang **Search Section** upang matukoy kung aling department ang gumagawa ng request.

3. **Itakda ang Deadline**: Gamitin ang **Date Needed** picker upang tukuyin kung kailan kinakailangan ang mga items o serbisyo.

4. **Magdagdag ng Items**: I-type ang **Item Code** o **Description** sa details section upang magdagdag ng items sa iyong request.

5. **I-finalize**: I-review ang listahan at i-click ang **Save** button.

### Paano Mag-edit ng Transaksyon

> ✔️ **Tip sa Pag-edit:** Maaari mong baguhin ang isang transaksyon hangga’t hindi pa ito napo-post o nakacancel.

1. **Pumasok sa Edit Mode**: I-click ang **Edit** button upang ma-unlock ang form.

2. **Gumawa ng Mga Pagbabago**: I-update ang item quantities, descriptions, o mga kinakailangang petsa.

3. **I-save o I-discard**:  
**Save Button:** Magiging active lamang habang nasa editing mode. I-click upang i-save ang mga pagbabago.  
**Undo Edit:** I-click ito upang kanselahin ang mga pagbabago at ibalik sa huling na-save na estado.

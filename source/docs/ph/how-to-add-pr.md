# Purchase Request

Ang **Purchase Request (PR)** sa MyxFin ay nagbibigay-daan sa mga user na humiling ng mga item o serbisyo na kailangan ng negosyo bago gumawa ng isang purchase order. Ang module na ito ay tumutulong sa pag-streamline ng proseso ng pagbili sa pamamagitan ng pag-track ng mga request, at pagtiyak na ang mga approval at records ay maayos na pinamamahalaan.

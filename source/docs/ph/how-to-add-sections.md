# Sections

Ang mga Section ay nagtatakda ng mga lokasyon ng inventory o mga kategorya na ginagamit sa mga inventory transaction gaya ng transfers, physical counts, at inventory adjustments. Nakakatulong ang mga ito upang masiguro ang tumpak na pagsubaybay at kontrol sa paggalaw ng stocks sa loob ng system.

---

### Paano Gumawa ng Bagong Section:

1. Pumunta sa **Master Data Files** 
2. I-click ang **Sections**
3. I-click ang **Create New (F1)**
4. Ilagay ang mga detalye ng bagong section. 
5. I-click ang **Save (F2)**

![Navigating to Section](/assets/img/section.png)

### Paano Mag-update ng Section:

1. Pumunta sa **Master Data Files**
2. I-click ang **Sections** 
3. Pumili ng section at i-click ang blue font text o ang Section Name. 
4. Ilagay ang mga bagong detalye ng section. 
5. I-click ang **Save (CTRL+S)**

![Navigating to Section](/assets/img/section1.png)

### Paano Gawing Inactive ang Section:

1. Pumunta sa **Master Data Files**
2. I-click ang **Sections**
3. I-click ang **yellow cycle icon** o ang **Inactive** button.

![Navigating to Section](/assets/img/section2.png)

# Suppliers Type

### Paano Magdagdag ng Bagong Suppliers Type:

1. Pumunta sa **Master Data Files**. 
2. I-click ang **Suppliers**.
3. I-click ang **Types**.
4. I-click ang **Create New (F1)**.
5. Ilagay ang mga detalye ng bagong suppliers type. 
6. I-click ang **Save (F2)**.

![Navigating to SupplierType](/assets/img/supptype.png) 

---

### Paano Mag-update ng Suppliers Type:

1. Pumunta sa **Master Data Files**. 
2. I-click ang **Suppliers**.
3. I-click ang **Types**.
4. I-click ang **Code**. 
5. I-update ang mga detalye ng supplier’s type.  
6. I-click ang **Save (F2)**.

![Navigating to SupplierType](/assets/img/supptype1.png) 

---

### Paano Gawing Inactive ang Suppliers Type:

1. Pumunta sa **Master Data Files**. 
2. I-click ang **Suppliers**.
3. I-click ang **Type**.
4. I-click ang **yellow cycle icon** o ang **Inactive** button.

* I-click naman ang **green cycle icon** o ang **Active** button para muling i-activate ang item.  

![Navigating to SupplierType](/assets/img/supptype2.png)

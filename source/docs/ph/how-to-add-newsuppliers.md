# Suppliers

Ang Suppliers module sa MyxFin ay ginagamit upang maayos na pamahalaan ang impormasyon ng mga vendor. Maaaring gumawa ng bagong supplier, i-update ang detalye ng existing suppliers, o i-inactivate ang mga suppliers na hindi na aktibo. Nakakatulong ito upang masiguro na tama at updated ang iyong accounts payable at purchasing records.

---

### Paano Gumawa ng Bagong Supplier:

1. Pumunta sa **Master Data Files**
2. I-click ang **Suppliers**
3. I-click ang **Master List**
4. I-click ang **Create New (F1)** 
5. Ilagay ang mga detalye ng bagong supplier. 
6. I-click ang **Save (F2)**

![Navigating to New Suppliers](/assets/img/supp.png)
![Navigating to New Suppliers](/assets/img/supp1.png)

> 💡 **Tandaan:** Sa initial setup, ang data para sa mga supplier ay ia-upload mula sa filled-out template na ibibigay ng mga kliyente.

---

### Paano Mag-update ng Supplier:

1. Pumunta sa **Master Data Files** 
2. I-click ang **Suppliers**
3. I-click ang **Master List**
4. Pumili ng supplier at i-click ang blue font text o ang Supplier Code. 
5. Kapag lumabas na ang mga detalye, i-click ang **Edit (CTRL+E)**
6. Ilagay ang mga bagong detalye ng supplier. 
7. I-click ang **Save (CTRL+S)**

![Navigating to CustomerGroup](/assets/img/supp3.png)    

![Navigating to CustomerGroup](/assets/img/supp4.png)    

---

### Paano Gawing Inactive ang Supplier:

1. Pumunta sa **Master Data Files**
2. I-click ang **Suppliers**
3. I-click ang **Master List**
4. I-click ang **yellow cycle icon** o ang **Inactive** button.

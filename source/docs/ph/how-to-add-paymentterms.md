# Payment Terms

Ang Payment Terms ay nagtatakda ng mga kondisyon sa pagbabayad na ginagamit sa mga transaksyon ng customer at supplier. Kinokontrol ng mga terms na ito ang pagkalkula ng due date, mga schedule ng pagbabayad, at ang aging sa loob ng ERP system.

### Paano Gumawa ng Bagong Payment Terms:

1. Pumunta sa **Master Data Files**
2. I-click ang **Accounting Parameters**
3. I-click ang **Payment Terms** 
4. I-click ang **Create New (F1)** 
5. Ilagay ang mga detalye ng Payment Terms. 
6. I-click ang **Save**

![Navigating to Payment Term](/assets/img/npayment.png) 

---

### Paano Mag-update ng Payment Term:

1. Pumunta sa **Master Data Files**
2. I-click ang **Accounting Parameters**
3. I-click ang **Payment Terms**
4. I-click ang **yellow button**
5. I-update ang mga detalye ng Payment Terms. 
6. I-click ang **Update**

![Navigating to Payment Term](/assets/img/npayment1.png) 

---

### Paano Gawing Inactive ang Payment Terms:

1. Pumunta sa **Master Data Files**
2. I-click ang **Accounting Parameters**
3. I-click ang **Payment Terms**
4. I-click ang **yellow cycle icon** o ang **Inactive** button.

I-click naman ang **green cycle icon** o ang **Active** button para muling i-activate ang item.

![Navigating to Payment Term](/assets/img/npayment2.png)

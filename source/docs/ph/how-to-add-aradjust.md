# AR Adjustment

Ang AR (Accounts Receivable) Adjustment ay nagbibigay-daan sa mga user na mag-correct, mag-update, o mag-reverse ng mga customer-related na transaksyon. Nakakatulong ito upang manatiling tama at maayos ang iyong records kapag may inaayos na pagkakamali sa invoices o payments.

> 💡 **Tandaan:** Tanging mga transaksyon na walang reference (o may cancelled na reference sa Receive Payment) lamang ang maaaring i-unpost.

---

### Paano Gumawa ng Bagong Transaksyon

1. **Buksan ang Form**  
I-click ang **Create New** button upang magsimula ng bagong adjustment.

2. **Piliin ang Reference Type** :  
Para sa **Sales Returns**: I-check ang Sales Return box bago i-click ang search button. Magge-generate ito ng automatic accounting entry (na maaari pa ring i-edit).  
Para sa **Sales Invoices**: I-uncheck ang Sales Return box, pagkatapos ay i-click ang **Search** button upang piliin ang reference Sales Invoice.

3. **Magdagdag ng Line Items**  
I-click ang **Add New Line** button sa details section upang maglagay ng adjustment entries.

4. **I-save ang Transaksyon**  
Kapag tama na ang lahat ng detalye, i-click ang **Save** button upang ma-finalize.

---

### Paano Mag-edit ng Transaksyon

Kung kailangan mong baguhin ang isang existing na adjustment, sundin ang mga hakbang na ito:

1. **I-activate ang Edit Mode**  
I-click ang **Edit** button upang ma-unlock ang form fields para sa pagbabago.

2. **Baguhin ang Mga Detalye**  
I-update ang kinakailangang amounts o account titles.

3. **I-save ang Mga Pagbabago**  
Habang nasa editing mode, magiging active ang **Save** button. I-click ito upang ma-apply ang iyong updates.

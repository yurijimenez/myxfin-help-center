# System Users

Ang mga System User ay nagtatakda ng mga awtorisadong account sa ERP system. Bawat user ay binibigyan ng partikular na access sa mga module at pahintulot (permissions) sa mga aksyon gaya ng view, add, edit, post, cancel, at print. Ang mga setting na ito ang nagkokontrol sa mga function na maaaring gamitin ng user upang masiguro ang seguridad at accountability sa system.

---

### Paano Gumawa o Magdagdag ng Bagong User:

1. Pumunta sa **Master Data Files**
2. I-click ang **System Users**
3. I-click ang **Create New (F1)**
4. Ilagay ang mga detalye para sa bagong user. 
5. I-click ang **Add User**

![Navigating to System User](/assets/img/sysuser.png)

---

### Paano I-set ang Access ng User:

1. Pumunta sa **Master Data Files**
2. I-click ang **System Users**
3. Pumili ng user at i-click ang **pencil icon** o ang Edit user's access. 
4. Piliin ang mga **category** kung saan magkakaroon ng access ang user. 
5. Kapag napili na ang category, lagyan ng check ang checkbox na tumutugma sa access ng user.   
6. Kung ang user ay may access sa lahat, i-tick lamang ang **CHECK ALL** 
7. I-click ang **Save (F2)**

![Navigating to System User](/assets/img/sysuser1.png)
![Navigating to System User](/assets/img/sysuser2.png)

---

### Paano I-reset ang Password ng User:

1. Pumunta sa **Master Data Files** 
2. I-click ang **System Users**
3. I-click ang **cycle icon** o ang **Reset** user password. 

![Navigating to System User](/assets/img/sysuser3.png)

--- 

### Paano Gawing Inactive ang User Access:

1. Pumunta sa **Master Data Files**
2. I-click ang **System Users** 
3. I-click ang **x icon** o ang **Inactive** user access. 

Upang muling i-activate ang inactive na user, i-click lamang ang **check icon** o ang **Activate** user access.

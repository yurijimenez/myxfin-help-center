# Bank Masterlist

Ang "bank" ay tumutukoy sa mga financial institution kung saan ang negosyo ay mayroong financial accounts, gaya ng checking accounts, savings accounts, at iba pang banking activities. Mahalaga ang mga bangko sa mga transaksyong pinansyal, pagproseso ng bayad, at sa pamamahala ng aspetong pinansyal ng sales at inventory. Nakakatulong din ito sa Cash Flow Management sa pamamagitan ng pag-track ng mga papasok at lalabas na pera para masiguradong sapat ang pondo ng negosyo.

### Paano Gumawa ng Bagong Bank:
1. Pumunta sa **Master Data Files**
2. I-click ang **Accounting Parameters**
3. I-click ang **Banks**
4. I-click ang **Create New (F1)**
5. Ilagay ang mga detalye ng bagong bangko. 
6. I-click ang **Save (F2)**

### Paano Mag-update ng Bank:

1. Pumunta sa **Master Data Files**
2. I-click ang **Accounting Parameters** 
3. I-click ang **Banks**
4. Pumili ng bangko at i-click ang blue font text o ang Bank Code. 

5. Kapag lumabas na ang bank details, i-click ang **Edit (CTRL+E)**

6. Ilagay ang mga bagong detalye ng bangko. 
7. I-click ang **Save (CTRL+S)**

### Paano Gawing Inactive ang Bank:

1. Pumunta sa **Master Data Files**
2. I-click ang **Accounting Parameters**
3. I-click ang **Banks**
4. I-click ang **yellow cycle icon** o ang **Inactive** button.

I-click naman ang **green cycle icon** o ang **Active** button para muling i-activate ang item.

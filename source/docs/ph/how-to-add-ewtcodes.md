# EWT Codes

Ang EWT Codes ay ginagamit para sa Expanded Withholding Tax classifications ng system. Ang mga EWT Codes na kasalukuyang available ay base sa mga pinaka-karaniwang ginagamit na BIR EWT codes at hindi pa kumakatawan sa buong listahan na inilabas ng BIR.

### Paano Gumawa ng Bagong EWT Code:

1. Pumunta sa **Master Data Files**. 
2. I-click ang **Accounting Parameters**.
3. I-click ang **EWT Codes**. 
4. I-click ang **Create New (F1)**. 
5. Ilagay ang mga detalye ng EWT Code. 
6. I-click ang **Add Detail**.

![Navigating to EWT](/assets/img/ewt.png) 

---

### Paano Mag-update ng EWT Code:

1. Pumunta sa **Master Data Files**. 
2. I-click ang **Accounting Parameters**.
3. I-click ang **EWT Codes**. 
4. Pumili ng EWT code at i-click ang blue font text o ang mismong EWT Code. 
5. Ilagay ang mga bagong detalye ng EWT Code. 
6. I-click ang **Update Detail**.

![Navigating to EWT](/assets/img/ewt1.png) 

---

### Paano Gawing Inactive ang EWT Code:

1. Pumunta sa **Master Data Files**. 
2. I-click ang **Accounting Parameters**.
3. I-click ang **EWT Code**. 
4. I-click ang **yellow cycle icon** o ang **Inactive** button.

* I-click naman ang green cycle icon o ang Active button para muling i-activate ang item. 

![Navigating to EWT](/assets/img/ewt2.png)

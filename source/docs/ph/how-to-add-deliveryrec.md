# Delivery Receipt

Ang Delivery Receipt (DR) ay ginagamit upang maitala ang aktwal na pagpapadala ng mga produkto sa iyong mga customer. Ito ay mahalagang hakbang bago makagawa ng final Sales Invoice.

> 💡 **Tandaan:** Tanging mga transaksyon na walang reference o may cancelled reference sa Sales Invoice lamang ang maaaring i-unpost.

---

### Paano Gumawa ng Bagong Transaksyon

1. **Simulan ang Entry**  
I-click ang **Create New** button.

2. **Maglagay ng Details (Pumili ng Paraan)**  
**Paraan A (Mabilis na Entry):** I-type ang **SO Number** sa reference field at pindutin ang **ENTER**. Awtomatikong lalabas ang customer name at lahat ng items.  
**Paraan B (Manual na Pagpili):** Hanapin muna ang **Customer**, pagkatapos ay i-click ang **SO (Insert)** button.

3. **Pumili mula sa Sales Order (Kung Paraan B)**  
Sa pop-up window, i-click ang partikular na **SO No.**, i-check ang mga items na ide-deliver, at i-click ang **Insert** upang mailagay sa main form.

4. **Barcode/Serial Entry (Optional)**  
Kung may barcode o serial number ang items, i-click ang **Barcode/Serial Insert** button sa details section, ilagay ang quantity, at i-click ang **Insert**.

5. **I-save**  
Suriin ang quantities upang matiyak na tugma ito sa actual na items, pagkatapos ay i-click ang **Save**

### Paano Mag-edit ng Transaksyon

1. **Pumasok sa Edit Mode**  
I-click ang **Edit** button upang ma-unlock ang transaction fields.

2. **Magdagdag ng Items**  
Habang nasa edit mode, maaari mong i-click ang **SO Insert** button upang magdagdag ng items mula sa existing Sales Order.

3. **Maglagay ng Item Remarks**  
Gamitin ang **Plus (+)** button sa loob ng details table upang magdagdag ng notes o remarks sa isang item.

4. **I-finalize**  
Kapag tapos na ang pagbabago, magiging active ang **Save** button. I-click ito upang ma-save ang updates.

# Customer Classification

Ang Customer Classification sa MyxFin ay ginagamit upang ikategorya ang mga customers para sa mas maayos na pamamahala at reporting. Nakakatulong ang module na ito upang ma-grupo ang mga customers batay sa type, region, o iba pang custom criteria, na nagpapadali sa sales, invoicing, at reporting processes.

---

### Paano Magdagdag ng Bagong Customer Classification:
1. Pumunta sa **Master Data Files**. 
2. I-click ang **Customers**.
3. I-click ang **Classification**.
4. I-click ang **Create New (F1)**.
5. Ilagay ang mga detalye ng bagong customer classification. 
6. I-click ang **Save (F2)**.

![Navigating to CustomerType](/assets/img/custcl.png)

---

### Paano Mag-update ng Customer Classification:
1. Pumunta sa **Master Data Files**. 
2. I-click ang **Customers**.
3. I-click ang **Classification**.
4. I-click ang **Class Code**.
5. I-update ang mga detalye ng customer classification. 
6. I-click ang **Save (F2)**.

![Navigating to Customer](/assets/img/custcl1.png)    

---

### Paano Gawing Inactive ang Customer Classification:

1. Pumunta sa **Master Data Files**. 
2. I-click ang **Customers**.
3. I-click ang **Classification**.
4. I-click ang **yellow cycle icon** o ang **Inactive** button.

* I-click naman ang **green cycle icon** o ang **Active** button para muling i-activate ang item.

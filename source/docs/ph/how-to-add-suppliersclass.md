# Suppliers Classification

### Paano Magdagdag ng Bagong Suppliers Classification:

1. Pumunta sa **Master Data Files**. 
2. I-click ang **Suppliers**.
3. I-click ang **Classification**.
4. I-click ang **Create New (F1)**. 
5. Ilagay ang mga detalye ng bagong supplier’s classification. 
6. I-click ang **Save (F2)**. 

![Navigating to SupplierClass](/assets/img/suppclass.png) 

---

### Paano Mag-update ng Suppliers Classification:

1. Pumunta sa **Master Data Files**. 
2. I-click ang **Suppliers**. 
3. I-click ang **Classification**. 
4. I-click ang **Code**. 
5. I-update ang mga detalye ng supplier’s classification. 
6. I-click ang **Save (F2)**.

![Navigating to SupplierClass](/assets/img/suppclass1.png) 

---

### Paano Gawing Inactive ang Classification:

1. Pumunta sa **Master Data Files**. 
2. I-click ang **Suppliers**. 
3. I-click ang **Classification**.
4. I-click ang **yellow cycle icon** o ang **Inactive** button.

* I-click naman ang **green cycle icon** o ang **Active** button para muling i-activate ang item.  

![Navigating to SupplierClass](/assets/img/suppclass2.png)

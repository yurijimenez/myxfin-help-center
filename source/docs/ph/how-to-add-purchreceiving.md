# Purchase Receiving

Ang Purchase Receiving module ay ginagamit upang i-dokumento ang pagdating ng mga produkto. Ang hakbang na ito ay nag-a-update ng iyong inventory levels at nagsisilbing basehan para sa Supplier's Invoice.

> 💡 **Tandaan:** Ang mga transaksyon ay maaari lamang i-unpost kung wala itong active references (o may cancelled references) sa **Suppliers Invoice** o **Purchase Return**.

---

### Paano Gumawa ng Bagong Transaksyon

![Navigating to PRec](/assets/img/purchrec1.png)

1. **Simulan ang Pagtanggap**  
I-click ang **Create New** button.

2. **Piliin ang Supplier**  
Hanapin at piliin ang **Supplier** bago magdagdag ng anumang items.

3. **Magdagdag ng Items (Dalawang Paraan)**  
**Paraan A: Manual Entry** → I-type ang detalye ng item at pindutin ang **ENTER** o i-click ang **Insert** button. Gamitin ang **Barcode/Serial** button kung may specific tracking numbers ang items.  
**Paraan B: PO Reference** → I-click ang **PO Insert** button. Sa pop-up window, i-click ang **PO No.**, i-check ang mga box para sa mga natanggap na items, at i-click ang **Insert**.

4. **Review at Linisin**  
Kung nagdagdag ng maling item, i-click ang **Delete** (Trash) icon sa details section.

5. **I-save**  
Kapag tumutugma na ang physical count sa screen, i-click ang **Save** button.

### Paano Mag-edit ng Transaksyon

> ✔️ **Tip sa Pag-edit:** Maaari mong baguhin ang transaksyon hangga’t hindi pa ito na-post o nakansela.

1. **Pumasok sa Edit Mode**  
I-click ang **Edit** button upang ma-unlock ang transaction details.

2. **Baguhin ang Records**  
I-update ang quantities o magdagdag ng nawawalang items.

3. **I-save o I-discard**  
**Save:** Nagiging active lamang habang nasa editing mode. I-click upang i-save ang pagbabago.  
**Undo Edit:** I-click upang kanselahin ang mga pagbabago at lumabas sa edit mode nang hindi nagse-save.

![Navigating to PR](/assets/img/pr4.png)

---

### Mga Tip sa Navigation

* **Back to Main:** Bumabalik sa list view nang hindi sine-save ang kasalukuyang transaksyon.  

* **Close Button:** Pagkatapos manu-manong mag-insert ng items, i-click ang **Close** upang bumalik sa main transaction form.

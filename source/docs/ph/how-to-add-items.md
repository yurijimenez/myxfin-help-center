# Item Masterlist

Ang "Items" ay tumutukoy sa mga indibidwal na produkto o kalakal na ibinebenta o pinamamahalaan ng isang negosyo bilang bahagi ng kanilang inventory. Ang bawat natatanging produkto na inaalok ng kumpanya ay kinakategorya at sinusubaybayan bilang isang "item" sa loob ng system. Mahalaga ang mga Item sa pamamahala ng inventory, pagproseso ng mga sales order, pagsubaybay sa stock levels, at pagbibigay ng tumpak na impormasyong pinansyal.

### Paano Gumawa ng Bagong Item:
1. Pumunta sa **Master Data Files** 
2. I-click ang **Items**
3. I-click ang **Master list**
4. I-click ang **Create New (F1)**
5. Ilagay ang mga detalye ng bagong item. 
6. I-click ang **Save (CTRL+S)**

### Paano Mag-update ng mga Item:

1. Pumunta sa **Master Data Files**
2. I-click ang **Items**
3. I-click ang **Master List**
4. Piliin ang item at i-click ang asul na text o ang Part No/Item Code. 
5. I-click ang **Edit (CTRL+E)**
6. Ilagay ang mga updated na detalye para sa item.  
7. I-click ang **Save (CTRL+S)**

### Paano i-inactive ang mga Item:

1. Pumunta sa **Master Data Files**
2. I-click ang **Items**
3. I-click ang **Master List**
4. I-click ang **yellow cycle icon** o **Inactive**
5. I-click ang **green cycle icon** o **Active** para muling i-activate ang item.

# Bills Payment

Ang Bills Payment module ay ginagamit upang bayaran ang iyong outstanding Accounts Payable. Ito ang huling hakbang kung saan pipiliin mo kung aling vouchers ang babayaran at pamamahalaan ang pag-iisyu ng tseke.

![Navigating to Bills Payment](/assets/img/billsp.png)

---

### Paano Gumawa ng Bagong Transaksyon

1. **Simulan ang Payment**:  
I-click ang **Create New** button upang magbukas ng bagong form.

2. **Piliin ang Supplier**:  
Hanapin at piliin ang **Supplier** mula sa dropdown list.

3. **Piliin ang Vouchers (AP List)**:  
Awtomatikong lalabas ang pop-up window. I-check ang box ng partikular na **AP No.** (Voucher) na nais mong bayaran, pagkatapos ay i-click ang **Insert** button.

4. **Piliin ang Payment Bank**:  
I-click ang **Blue Search** button upang piliin ang source bank account. Para sa cheque payments, ipapakita rin dito ang kasalukuyan at reserved na check numbers.

5. **Pamahalaan ang Checks (Optional)**:  
Kung kailangan mong mag-reserve o mag-void ng specific na check number, i-click ang **Red Button (Reserve/Void Check)**. Ilagay ang User ID at Password ng authorized staff, magbigay ng valid na dahilan, at i-click ang **Proceed**.

6. **Magdagdag ng Manual Entries**:  
Kung kailangan ng karagdagang detalye o manual accounting entries, i-click ang **Add Line** button sa details section.

7. **I-save**:  
Suriin ang iyong totals at i-click ang **Save** button upang ma-finalize ang payment.

---

### Paano Mag-edit ng Transaksyon

1. **Pumasok sa Edit Mode**:  
I-click ang **Edit** button upang ma-unlock ang fields ng existing payment.

2. **Baguhin ang Details**:  
I-update ang payment date o magdagdag ng iba pang vouchers kung kinakailangan.

3. **I-save o I-cancel**:  
I-click ang **Save** upang ma-apply ang changes habang nasa editing mode, o i-click ang **Undo Edit** upang kanselahin at lumabas nang hindi sine-save.

4. **I-review ang Accounting**:  
Pagkatapos mag-save, i-click ang **Accounting Entry** button upang makita ang journal entries na ginawa para sa payment na ito.

![Navigating to Bills Payment](/assets/img/billsp3.png)

---

### Navigation Shortcuts

* **Back to Main:** I-click upang bumalik sa list view nang hindi sine-save ang kasalukuyang ginagawa.  

* **New Button:** Gamitin ito upang agad makapagsimula ng panibagong payment transaction nang hindi na bumabalik sa dashboard.

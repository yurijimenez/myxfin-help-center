# Suppliers Groupings

### Paano Gumawa ng Bagong Groupings:

**A. Palitan ang Pangalan ng Group Label**

1. Pumunta sa **Master Data Files**
2. I-click ang **System Settings**
3. Pumunta sa **Parameters Tab**
4. Sa ilalim ng Suppliers Groupings, ilagay ang label para sa bawat grupong kailangan mo. 
5. Pindutin ang Enter pagkatapos i-type ang label; may lalabas na check mark sa tabi ng Group para ipakita na ang pagbabago ay naka-save na. 

![Navigating to SupplierGroup](/assets/img/suppgroup.png) 

**B. Magdagdag ng Group Options**

1. Pumunta sa **Master Data Files**
2. I-click ang **Suppliers**
3. I-click ang **Groupings**
4. I-click ang **Create New (F1)**
5. Ilagay ang mga detalye ng bagong option. 
6. I-click ang **Add Details**

![Navigating to SupplierGroup](/assets/img/suppgroup1.png)

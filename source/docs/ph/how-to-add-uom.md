# Units Of Measure

![Navigating to Uom](/assets/img/uom.png)

### Paano Gumawa ng Bagong Unit of Measure:
1. Pumunta sa **Master Data Files** 
2. I-click ang **Items** 
3. I-click ang **Unit of Measure**
4. I-click ang **Create New (F1)**
5. Ilagay ang mga detalye ng bagong UOM. 
6. I-click ang **Add Details**

---

### Paano Mag-update ng Unit of Measure:

1. Pumunta sa **Master Data Files**
2. I-click ang **Items**
3. I-click ang **Unit of Measure**
4. Piliin ang **uom** at i-click ang **blue font text** o ang **UOM Code**. 
5. Ilagay ang mga binagong detalye para sa UOM. 
6. I-click ang **Update** Detail.

![Navigating to Uom](/assets/img/uom1.png)

---

### Paano Gawing Active/Inactive ang Unit of Measure:

1. Pumunta sa **Master Data Files**
2. I-click ang **Items**
3. I-click ang **Unit of Measure** 
4. I-click ang **Active** (Berde) / **Inactive** (Dilaw) na icon.  

![Navigating to Uom](/assets/img/uom2.png)

# Salesman

Ang isang **"salesman"** ay karaniwang tumutukoy sa isang indibidwal o grupo na responsable sa pagbebenta ng mga produkto o serbisyo sa mga customer. Ang tungkulin ng isang salesman ay maaaring magbago depende sa pangangailangan ng organisasyon.

--- 

### Paano Gumawa ng Bagong Salesman:

1. Pumunta sa **Master Data Files**
2. I-click ang **Salesman**
3. I-click ang **Create New**
4. Ilagay ang mga detalye sa lahat ng kinakailangang fields. 
5. I-click ang **Save**

# Purchase Return

Ang **Purchase Return (PR)** ay nagbibigay-daan sa mga user na ibalik ang mga biniling produkto sa mga vendor sa loob ng MyxFin. Ang module na ito ay ginagamit upang itama o i-reverse ang mga nakaraang transaksyon sa pagbili, upang matiyak na tumpak ang iyong inventory at records sa accounts payable.

---

![Pag-navigate sa Pret](/assets/img/purchret.png)

> 💡 **Tandaan:** Tanging ang mga transaksyong walang reference o may kanseladong reference sa AP Adjustment ang maaaring i-void.

---

### Paggawa ng Bagong Transaksyon:

![Pag-navigate sa Pret](/assets/img/purchret1.png)
![Pag-navigate sa Pret](/assets/img/purchret2.png)

**Mag-search** muna ng supplier bago magdagdag ng mga item.

**Ilagay** ang qty (dami) at i-click ang insert.

**PR (insert)** > I-click ito para magdagdag ng item mula sa receiving.

- **I-click** ang RR No. 

- **Pumili** mula sa listahan ng mga item sa pamamagitan ng pag-check sa box sa kaliwang bahagi ng item.

- **I-click** ang Insert.

---

### Pag-edit ng Transaksyon:

![Pag-navigate sa Pret](/assets/img/purchret3.png)
![Pag-navigate sa Pret](/assets/img/purchret4.png)

**I-click** ang edit para ma-activate ang form para sa pag-edit.

**RR Insert** > I-click para magdagdag pa ng mga item base sa Receiving (RR).

**Pulang Button (Undo Edit)** > I-click para ikansela ang pag-edit.

Kapag ang form ay nasa editing mode, ma-aactivate ang **Save button**.

# Purchase Return

Ang Purchase Return ay ginagamit upang ibalik ang mga produkto sa vendor. Ang module na ito ay nagko-korek sa mga naunang purchase transactions, tinitiyak na tama ang iyong inventory levels at Accounts Payable records.

> 💡 **Tandaan:** Ang mga transaksyon ay maaari lamang i-void kung wala itong active references (o may cancelled references) sa **AP Adjustment**.

---

### Paano Gumawa ng Bagong Transaksyon

1. **Simulan ang Return**  
I-click ang **Create New** button upang buksan ang form.

2. **Piliin ang Supplier**  
Hanapin at piliin ang pangalan ng **Supplier** bago magdagdag ng anumang items.

3. **Buksan ang Item Reference**  
I-click ang **PR (Insert)** button. Magbubukas ito ng hiwalay na pop-up window upang kunin ang items mula sa iyong Receiving Reports.

4. **Piliin ang Reference**  
Sa loob ng pop-up, i-click ang specific na **RR No.** kung saan nanggaling ang items na ibabalik.

5. **Piliin ang Items na Iba-back**  
I-check ang mga box sa kaliwa para sa mga items na ibabalik.

6. **I-insert sa Main Form**  
I-click ang **Insert** button sa ilalim ng pop-up. Ililipat nito ang items sa main transaction details.

7. **I-save**  
Suriin ang return quantities sa main form at i-click ang **Save** button.

### Paano Mag-edit ng Transaksyon

1. **Pumasok sa Edit Mode**  
I-click ang **Edit** button upang ma-unlock ang form.

2. **Magdagdag ng Karagdagang Items**  
I-click ang **RR Insert** upang muling buksan ang item selection window kung kailangan mong magdagdag ng items mula sa ibang receiving record.

3. **Tanggalin ang Items**  
Gamitin ang **Delete** button sa tabi ng anumang line item upang alisin ito sa return list.

4. **I-save o I-discard**  
**Save Button:** Nagiging active lamang habang nasa editing mode. I-click upang i-save ang pagbabago.  
**Red Button (Undo Edit):** I-click upang kanselahin ang kasalukuyang pagbabago.

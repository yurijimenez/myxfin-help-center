# Sales Pricelist

Ang Sales Price List, na kilala rin bilang **Price Matrix**, ay naglalaman ng mga structured pricing rules para sa mga items. Sinisiguro nito na consistent at tama ang presyo sa lahat ng sales transactions sa loob ng ERP system.

---

### Paano Makita at Magdagdag ng Bagong PM Versions:

1. Pumunta sa **Master Data Files**. 
2. I-click ang **Price List**.
3. I-click ang **Sale Pricelist**.
4. I-click ang **PM Version**.
5. I-click ang **Add New**.
6. Ilagay ang **new Version Code** at **Version Description**. 

* **Para mag-delete** ng partikular na PM Version, i-click lamang ang delete.

* **Para mag-update**, ilagay lamang ang mga bagong detalye sa Version Description. Tanging ang Version Description lamang ang maaaring baguhin.

7. I-click ang **Save Details**. 

![Navigating to SP](/assets/img/sp.png)

---

### Paano Magdagdag ng Bagong Price Matrix:

1. Pumunta sa **Master Data Files**. 
2. I-click ang **Price List**.
3. I-click ang **Sale Pricelist**.
4. I-click ang **Create New (F1)**.
5. Piliin ang **PM Version**.
6. I-click ang **Proceed**.
7. Ilagay ang Description at Effectivity Date. 
8. Ilagay ang product code o description para piliin ang item. 
9. Ilagay ang price matrix ng item. 
10. I-click ang **Save**.
11. I-click ang **Post**.

---

### Paano Mag-update ng Price Matrix:

1. Pumunta sa **Master Data Files**. 
2. I-click ang **Price List**.
3. I-click ang **Sale Pricelist**.
4. Piliin ang PM Batch No. at i-click ang blue text icon. 
5. I-click ang **Edit**.
6. Ilagay ang mga bagong detalye ng price matrix. 
7. I-click ang **Save (CTRL+S)**.

* Kapag ang status ay **Posted** na, hindi na ito maaaring i-edit.

# AP Adjustment

Ang **AP (Accounts Payable) Adjustment** ay nagbibigay-daan sa mga user na itama, i-update, o i-reverse ang mga transaksyong may kaugnayan sa vendor sa MyxFin. Ang feature na ito ay kapaki-pakinabang sa pag-aayos ng mga pagkakamali sa mga bill, pagbabayad, o credits, upang matiyak na tumpak ang iyong mga records sa accounts payable. Maaaring gumawa ang mga user ng mga bagong adjustment entries, i-edit ang mga kasalukuyang transaksyon, o i-unpost ang mga maling entry para mapanatili ang tumpak na financial reporting.

![Pag-navigate sa Adjustments](/assets/img/adjustment.png)
*Figure 1: Adjustment.*

> 💡 **Tandaan:** Tanging ang mga transaksyong walang reference o may kanseladong reference sa Receive Payment ang maaaring i-unpost.

# Customer Type

Ang Customer Type sa MyxFin ay ginagamit upang ikategorya ang mga customers batay sa partikular na attributes o pangangailangan ng negosyo. Nakakatulong ang module na ito upang mas maayos na pamahalaan ang customers, subaybayan ang transactions, at gumawa ng reports.

---

### Paano Gumawa ng Bagong Customer Type:
1. Pumunta sa **Master Data Files**. 
2. I-click ang **Customers**.
3. I-click ang **Types**.
4. I-click ang **Create New (F1)**.
5. Ilagay ang mga detalye ng bagong customer type. 
6. I-click ang **Save (F2)**.

![Navigating to CustomerType](/assets/img/custype.png)

---

### Paano Mag-update ng Customer Type:
1. Pumunta sa **Master Data Files**. 
2. I-click ang **Customers**.
3. I-click ang **Types**.
4. I-click ang **Type code**.
5. I-update ang mga detalye ng customer type. 
6. I-click ang **Update Detail**.

![Navigating to Customer](/assets/img/custype1.png)

---

### Paano Gawing Inactive ang Customer Type:

1. Pumunta sa **Master Data Files**. 
2. I-click ang **Customers**.
3. I-click ang **Types**.
4. I-click ang **yellow cycle icon** o ang **Inactive** button.

* I-click naman ang **green cycle icon** o ang **Active** button para muling i-activate ang item.

# AP Adjustment

Ang AP (Accounts Payable) Adjustment ay ginagamit upang baguhin o itama ang mga natitirang balanse na utang sa mga supplier. Tinitiyak nito na ang iyong AP Ledger ay tumutugma sa aktwal mong obligasyon matapos ang discounts, corrections, o returns.

> 💡 **Mahalagang Panuntunan:** Kailangan munang gumawa ng **Supplier’s Invoice** sa buong halaga nito bago gumawa ng adjustment. Ang adjustment na ito ay makikita sa **APV (Accounts Payable Voucher)**.

---

### Paano Gumawa ng Bagong Transaksyon

1. **Simulan ang Entry**:  
I-click ang **Create New** button.

2. **Piliin ang Supplier**:  
Ilagay muna ang pangalan ng **Supplier**. Hindi gagana nang maayos ang search tools hangga’t walang napipiling supplier.

3. **I-link ang Reference**:  
I-click ang **Search (Magnifying Glass)** button upang makita ang mga existing records.

4. **Piliin ang Reference No.**:  
Sa pop-up, i-click ang partikular na **PR No.** (Purchase Receipt) o **SI No.** (Supplier Invoice) na kailangang i-adjust.

5. **Itakda ang Accounting Details**:  
Sa Details section, ilagay ang tamang **Account Title** o **Account No.** para sa adjustment (hal., Purchase Discounts o Returns).

6. **I-save**:  
Suriin ang adjusted totals at i-click ang **Save** button.

### Paano Mag-edit ng Transaksyon

> 💡 **Tandaan:** Tanging mga transaksyon na walang reference (o may cancelled na reference sa AP Voucher) lamang ang maaaring i-unpost o i-edit.

1. **Hanapin ang Record**:  
Sa main dashboard, i-click ang partikular na **Transaction No.** na nais mong baguhin.

2. **I-activate ang Edit Mode**:  
I-click ang **Edit** button upang ma-unlock ang fields.

3. **I-update at I-save**:  
Kapag naitama na ang Account Titles o amounts, magiging active ang **Save** button. I-click ito upang ma-finalize ang iyong changes.
